Enrolla SDKs are available in multiple languages to allow you to integrate Enrolla into your production environment.

### Functionality

- **Evaluate features** - provide a *feature* `key` and a customer `organization_id` and get the resulting feature value.
- **Evaluate secrets** - provide a *secret* `key` and a customer `organization_id` and get the decrypted secret value.
- **Management API** - Use the SDK to create customers, update customer features/secrets, and more.

### Features

<Info>
  The SDKs pull all relevant customer data for your application and store it in
  a local cache. This allows you to access the data with zero latency and
  without the need to make any awaited API calls.
</Info>

- Smart caching of data for zero-latency access time.
- Automatic data synchronization with the Enrolla server or your backend of choice.
- Management API functionality 

The SDKs provide an easy to use API for evaluating feature/secret values for a given customer. See the language specific documentation for more details.
