---
title: 'Databases'
---

If you currently store your customer configurations or feature data in a database and want to import that data into Enrolla, we have you covered.

### How to Import data from your database

1. Navigate to the Integrations page in the Enrolla Dashboard. [Take me there now](https://app.enrolla.io/integrations).
2. Click on relevant database integration.
3. Follow the on-screen instructions. You will need to provide connection details and schema information.

### Supported Databases

- MongoDB
- PostgreSQL
